Research / Automation

AI News Monitor

Automatically collects AI vendor updates via RSS, filters for what matters, and delivers summaries to email and spreadsheet — 3 times a day.

Overview

Frequency

3 times a day, fully automated

Sources

New models, features & research from AI vendors

Output

Gmail digest + Google Sheets log

Problem & Solution

Before

Manually checking OpenAI, Anthropic, Google DeepMind and more every day led to missed updates and wasted time on information that didn't matter.

After

The agent crawls RSS feeds automatically, filters for important articles, summarizes them, and sends an email digest. Only what's worth reading lands in your inbox.

How It Works

01 Fetch new articles from multiple RSS feeds and extract only items published since the last run
02 Pass titles and summaries to AI, which selects only important articles (new models, features, pricing changes, research), assigns a category and importance score, and summarizes each in 1–2 lines
03 Send results as an HTML email to Gmail and append each item to a Google Sheet — all triggered automatically by cron 3 times a day

Background

AI moves fast, and I kept missing important updates. "I check this every day anyway — I should automate it." Sources, filtering rules, and the AI model are all in a config file, so adding a new topic takes seconds.

Tech Stack

Python Gemini API Google Sheets API Gmail SMTP RSS / feedparser cron (Hetzner VPS) Vibe Coding
